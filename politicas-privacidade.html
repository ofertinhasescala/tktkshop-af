🔒 Políticas de Privacidade

Air Fryer TikTok Shop - Proteção de Dados e Privacidade

📅 Última atualização: Janeiro de 2025
⚖️ Conformidade: Lei Geral de Proteção de Dados (LGPD) - Lei nº 13.709/2018

1. Informações Gerais

A presente Política de Privacidade tem por objetivo esclarecer como coletamos, usamos, armazenamos e protegemos suas informações pessoais quando você utiliza nosso site para adquirir produtos Air Fryer através do TikTok Shop.

🛡️ Compromisso: Respeitamos sua privacidade e estamos comprometidos em proteger seus dados pessoais em conformidade com a LGPD.

2. Dados Coletados

Dados fornecidos diretamente por você:

  • Dados de identificação: Nome completo, CPF
  • Dados de contato: E-mail, telefone, endereço
  • Dados de pagamento: Informações de cartão de crédito (processadas por terceiros seguros)
  • Dados de entrega: Endereço de entrega, preferências de horário

Dados coletados automaticamente:

  • Dados de navegação: Páginas visitadas, tempo de permanência, cliques
  • Dados técnicos: Endereço IP, tipo de dispositivo, navegador
  • Cookies: Preferências do usuário, carrinho de compras
  • Dados de analytics: TikTok Pixel para otimização de campanhas

3. Finalidade do Tratamento

Utilizamos seus dados pessoais para as seguintes finalidades legítimas:

  • ✅ Processamento de pedidos: Confirmação, pagamento e entrega do produto
  • ✅ Atendimento ao cliente: Suporte técnico, dúvidas e reclamações
  • ✅ Cumprimento legal: Emissão de notas fiscais, registros contábeis
  • ✅ Marketing (com consentimento): Envio de ofertas e novidades
  • ✅ Melhoria do serviço: Análise de performance e experiência do usuário
  • ✅ Segurança: Prevenção de fraudes e atividades suspeitas

4. Seus Direitos (Art. 18 da LGPD)

Como titular dos dados, você possui os seguintes direitos:

  • 📋 Confirmação: Saber se tratamos seus dados
  • 👁️ Acesso: Obter cópia dos seus dados
  • ✏️ Correção: Atualizar dados incompletos ou incorretos
  • 🗑️ Exclusão: Solicitar remoção dos dados desnecessários
  • 📤 Portabilidade: Transferir dados para outro fornecedor
  • 🚫 Oposição: Opor-se ao tratamento baseado em interesse legítimo
  • ❌ Revogação: Retirar consentimento a qualquer momento
📞 Como exercer seus direitos: Entre em contato através dos canais disponíveis na seção "Fale Conosco" ou pelo e-mail: privacidade@airfryershop.com

5. Segurança dos Dados

Implementamos medidas técnicas e administrativas adequadas para proteger seus dados:

  • 🔐 Criptografia: Dados sensíveis criptografados (HTTPS/SSL)
  • 🛡️ Controle de acesso: Acesso restrito apenas ao necessário
  • 📊 Monitoramento: Logs de acesso e atividades suspeitas
  • 🔄 Backups seguros: Cópias de segurança regulares
  • 🎓 Treinamento: Equipe capacitada em proteção de dados

6. Compartilhamento de Dados

Seus dados podem ser compartilhados apenas nas seguintes situações:

  • Processadores de pagamento: Para validação e processamento seguro
  • Transportadoras: Para entrega dos produtos
  • Plataforma TikTok Shop: Para gestão de pedidos e analytics
  • Autoridades fiscais e regulatórias: Quando exigido por lei

7. Retenção de Dados

Mantemos seus dados pelo tempo necessário para:

  • Dados de pedidos: 5 anos (obrigação fiscal)
  • Dados de marketing: Até revogação do consentimento
  • Dados de navegação: 12 meses
  • Logs de segurança: 6 meses

8. Cookies e Tecnologias Similares

Utilizamos cookies para melhorar sua experiência:

  • Essenciais: Funcionamento básico do site
  • Performance: Analytics e métricas de uso
  • Funcionais: Lembrar preferências do usuário
  • Marketing: TikTok Pixel para campanhas direcionadas

Você pode gerenciar cookies através das configurações do seu navegador.
